# Änderungsverlauf

Hier findest du eine Liste der Änderungen an der Applikation.

## Version 1.0.0

2024-01-09

**Neue Funktionen:**

- Auf "Mein Profil" kannst du nun bei Bedarf dein Konto löschen
- Liste von Praktikanten mit Kontaktdaten
- Du kannst nun die Profil Seite von jedem Praktikanten und jedem Trainer aufrufen. Klicke dafür auf den Namen oder das Profilbild in der Praktika-, Praktikanten- oder Trainer-Seite. Auch bei den Praktika Anmeldungen kannst du auf den Benutzernamen klicken, um das Profil von Praktikanten und deren Kontaktdaten zu sehen.

**Änderungen:**

- "Mein Profil" Link zur Navigation hinzugefügt
- Neues Design für "Mein Profil"
- Neues Design für Trainer:innen Seite
- Hilfe Text aus der Navigation entfernt
- Das Versenden von Einladungs-E-Mails beim Einladen von Benutzern ist jetzt optional.
- Link zur Registrierung auf Login Seite hinzugefügt
