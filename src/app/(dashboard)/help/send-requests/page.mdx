# Praktika Anfragen senden

Um die Kommunikation strukturierter und einfacher zu machen können Benutzer über den Menüpunkt "Trainer:innen", und Trainer:innen Profile Praktika Anfragen schicken. Du kannst Profile öffnen, indem du auf das Bild eines Benutzers drückst, oder auf den Namen auf der "Trainer:innen" Seite.

## Wie sende ich eine Anfrage?

Ein Klick auf den Button "Praktikum anfragen" auf Desktop Geräten öffnet den Dialog dafür. Auf mobilen Geräten kannst du das Menü mit den drei Punkten öffnen, und dort "Praktikum anfragen" drücken.

Du kannst dann eine optionale Nachricht eingeben und auf "Anfrage senden" drücken.

## Was passiert mit meiner Anfrage?

Der/die Trainer:in erhält ein E-Mail mit deiner Nachricht, und deinen hinterlegten Kontaktdaten. Um die Kommunikation zu vereinfachen, solltest du deshalb Telefonnummer in deinem Profil ausfüllen.

Wenn ein/e Trainer:in dann ein Praktikum auf der App erstellst, wirst du über eine automatisierte E-Mail benachrichtigt.

## Warum wird statt "Praktikum anfragen" der Text "Anfrage gesendet" angezeigt?

Damit du denselben Trainer:innen nicht täglich eine E-Mail schickst, gibt es eine Wartezeit von 7 Tagen, bis du noch eine Anfrage senden kannst.
