# App am Handy "installieren"

Da die App nur eine Webseite und keine richtige App ist, kann sie nicht aus den App Stores installiert werden (wäre auch ziemlich teuer). Stattdessen kannst du die App auf deinem Handy speichern, sodass sie wie eine App aussieht und sich auch so verhält.

## iOS

[Video Anleitung](https://www.loom.com/share/aa9b56462afe475b8c88fcd0a8aa573a?sid=6145c8a0-e2dd-4f6c-b9f7-73eaa50e8679)

- Öffne Safari (oder den Browser deiner Wahl) auf deinem iPhone und öffne [https://thl-praktika.vercel.app](https://thl-praktika.vercel.app).
- Drücke auf den Teilen Button <ShareIcon className="inline-block" /> (in der Mitte vom unteren Bildschirmrand), wähle "Zum Home-Bildschirm" und speichere die Webseite als App.

## Android

- Öffne auf deinem Android-Gerät Chrome Chrome.
- Rufe [https://thl-praktika.vercel.app](https://thl-praktika.vercel.app) auf.
- Tippe auf Installieren.
- Folge der Anleitung auf dem Bildschirm.

[Anleitung auf support.google.com](https://support.google.com/chrome/answer/9658361?hl=de&co=GENIE.Platform%3DAndroid&oco=0)
